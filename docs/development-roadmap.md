# Development Roadmap

## Milestone 1: Foundation

- Bootstrap Rust and TypeScript workspaces.
- Define schemas and examples for goals, policies, agent capabilities, eval reports, and release capsules.
- Add CLI validation for goals and policies.
- Add SDK event, metric, and feedback APIs.
- Add local runtime ingestion and signal summarization.
- Add eval runner and release capsule verification.

## Milestone 2: Demonstrable Loop

- Generate agent task briefs from local signals.
- Evaluate a patch through configured checks.
- Produce a release capsule for a demo artifact.
- Verify update metadata before installation.

## Milestone 3: Product Surface

- Build Adaptive Notes.
- Add a read-only dashboard for goals, signals, tasks, eval reports, and release capsules.
- Add plugin manifests and sandboxed adapter examples.
